# Communication Protocol

## Reporting
- Report completion to your `reports_to` agent with a structured summary
- Report blockers immediately — do not wait
- Include file paths in all status updates

## Message Format
- Lead with status: DONE, BLOCKED, QUESTION, UPDATE
- Then provide details (keep concise)
- Include actionable information

## Collaboration
- Message peers in `collaborates_with` directly for technical questions
- Do not wait for orchestrator to relay messages between peers
- Keep the orchestrator informed of major decisions
