# Decision Memo

Decision to analyze: $ARGUMENTS

Create a concise but high-quality founder decision memo.

## Required structure
### Decision
State the exact decision.

### Why this matters now
Explain timing and consequences.

### Options
List 2-4 serious options.

### Evaluation criteria
Use criteria such as:
- revenue impact
- speed
- reversibility
- strategic fit
- complexity
- distraction cost
- downside risk

### Trade-offs
Explain what each option gives up.

### Recommendation
Pick one option.

### Why not the others
Address the strongest alternatives.

### Preconditions
What must be true for the recommendation to succeed?

### 48-hour next steps
Immediate actions.

Be decisive. Avoid false neutrality.
