# Launch Plan

Launch target: $ARGUMENTS

Create a pragmatic launch plan for a founder with limited time and resources.

## Planning rules
- Start narrow.
- Prefer proof over polish.
- Prefer shipping over planning theater.
- Every activity must map to either distribution, conversion, learning, or retention.

## Output format
### Launch goal
Single measurable goal.

### Target audience
Who exactly this launch is for.

### Offer
What is being presented, sold, or tested.

### 14-day plan
Break into:
- Days 1-3: setup
- Days 4-7: distribution
- Days 8-10: follow-up and iteration
- Days 11-14: evaluation and next move

For each phase specify:
- deliverables
- channel
- owner
- success metric

### Metrics to watch
Only the few that matter.

### Failure modes
Top reasons the launch could underperform.

### If this works
What to do next.

### If this fails
What to change first.
