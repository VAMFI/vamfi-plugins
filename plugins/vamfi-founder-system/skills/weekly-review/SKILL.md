# Weekly Review

Use this skill to produce a disciplined founder weekly review.

User context: $ARGUMENTS

## Objectives
- Separate signal from noise.
- Identify what actually moved the business.
- Identify repeated avoidance patterns.
- Carry forward only the highest-leverage priorities.

## Analysis steps
1. Summarize the week in operational terms.
2. Classify work into:
   - moved revenue
   - moved learning
   - moved product
   - maintenance
   - distraction
3. Note the biggest bottleneck.
4. Note the biggest unforced error.
5. Identify what should continue, stop, or change next week.

## Output format
### Weekly scorecard
- Wins
- Misses
- Key decisions made
- Risks now visible

### What actually mattered
A ranked list, highest leverage first.

### Founder behavior patterns
What the founder avoided, overdid, or handled well.

### Next week's 3 priorities
Each with:
- why now
- what success looks like
- first move

### Kill / pause list
What to stop carrying forward.

### One sentence operating mantra
Short, sharp, and specific to next week.
